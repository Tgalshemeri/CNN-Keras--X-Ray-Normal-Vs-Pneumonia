# CNN Keras - X-Ray Normal Vs Pneumonia
An Example of CNN using Keras to Diagnose Pneumonia with 93.47% Accuracy.
The dataset is used from [Kaggle](https://www.kaggle.com/paultimothymooney/chest-xray-pneumonia/).
